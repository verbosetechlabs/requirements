Requirements

Welcome to Verbose TechLabs LLP, we are looking forward to serve you. Before we can start working on your awesome project, there are few requirements that you need to fulfill. Here we have list down all the requirements and steps to follow to arrange them. Kindly go through all of them to speed up the installation process.

Please fill the below form:

In Depth

Application Details

  • App Name: Name of your application, this name will be visible on Play Store, App Store and Admin Panel.
  • Logo Source File: Source file of your logo for all apps. You can share source file in any of these formats: .psd/.cdr/.ai
  • Domain Name: Where you want to host backend of your application.

Server

You need to provide the webserver where we will host the backend of your application. To speed up the installation process and to recieve future updates in smoother way, it is suggested to provide AWS.

  • AWS: It comes with 12 month free trial. While your app in development and has low number of users, it is the best option to choose, see here. If you are providing us the existing instance of server, please make sure you give us the server with following specification:
    • Ubuntu 18.04 or greater
    • User with sudo access
    • Mysql 5.7 or greater
    • Php 7.2 or greater

Also, we need to use https in order to comply with Google Play Store and Apple App Store latest policies. Following are the instructions for https:

  • You will to need to provide domain/subdomain name where you want to host Admin/backend of the application.
  • You will to need add an ‘A record’ in your DNS with the IP Address provided by us.
  • We will install SSL using ‘Let’s Encrypt’ on your server.

Mobile Application

Android

You need to purchase developer account from Google which will cost you USD 25(One time payment). After the purchase you need to share the Access to support@verbosetechlabs.com with Owner permission

Apple

You need to purchase developer account from Apple which will cost you $99 per year. After the purchase you need to share the credentials. Apple requires verification at the time of login, please be available for OTP verification. If you have business account, you can send an invite at support@verbosetechlabs.com from your Apple Account.
